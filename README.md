# Code Quality Control (CQC)

This code is for the research paper: "Code Quality Control by Bug Report Classification"

### Execution Environment

We executed the codes in the Jupyter Notebook environment of Google Colaboratory Service, 
so you need to upload the "00- My Data" and "01- Jupyter Notebook" folders to your Google Drive before execution.

### Requirements

Word2Vec pre-trained vectors can be downloaded from: https://code.google.com/archive/p/word2vec/.

### Dependencies

Python 3.7.13

PyTorch 1.11.0+cu113

torchvision 0.12.0+cu113

matplotlib 3.5.2

nltk 3.7

enlighten 1.10.2

numpy 1.21.6

seaborn 0.11.2
